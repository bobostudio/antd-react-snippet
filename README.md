# Antd React Snippet

高效开发 Ant Design + React + Ahooks 项目的 VS Code 代码片段插件。

## 特性

- 支持 ProTable 等常用 antd 组件片段
- 适配 TypeScript/TSX/JSX
- 适合企业级中后台开发

## 安装

1. 在 VS Code 插件市场搜索 `antd-react-snippet` 并安装
2. 或下载 .vsix 文件后手动安装

## 使用方法

- 在 `.tsx`/`.jsx`/`.ts` 文件中输入片段前缀（如 `rft`），按下 Tab 键即可展开代码模板

### 主要片段

| 前缀 | 说明                   |
| ---- | ---------------------- |
| rft  | 缓存 ProTable 列表模板 |

#### 示例：ProTable 缓存列表

```tsx
rft;
```
